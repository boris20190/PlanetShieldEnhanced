# PlanetShieldEnhanced

Enhances the Planetary Shield system for Dyson Sphere Program.

## Features

### Planetwide Shield Coverage
A single shield generator covers the entire planet surface. No need to build multiple generators.

### Ground Defense Shield
The planetary shield protects ground buildings from dark fog ground unit attacks. Damage is absorbed by the shield's energy pool using the same energy rate as space defense.

## Requirements
- Dyson Sphere Program (with Dark Fog combat enabled)
- BepInEx

## Installation
Place `PlanetShieldEnhanced.dll` in `BepInEx/plugins/PlanetShieldEnhanced/`.

> If you have **PlanetwidePlanetaryShieldGenerator** installed, disable it to avoid conflicts.
